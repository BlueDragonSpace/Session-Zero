Rules of Play


1. Create and ship anything related to TTRPGs (like D&D!)

This can be anything from a virtual battle map, a dice roll calculator / automator, combat timer, dungeon / player / npc generator, spellbook sorter, technical cosplay
...or literally anything else you can think up!

2. Track at least 5 hours using Hackatime and GitHub

Upload your project files on GitHub on an open-source repositoryHackatime is Hack Club's automatic coding time tracking tool! If you'd like to use an alternative time tracking tool, like Lapse, that'll work too!

3. Ship a finished project, get free dice!

Put your final polish on a project, then show it to the world!
We'll send a grant by HCB to buy any dice set you'd like!
